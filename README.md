# YouTube Rupiah Sentiment Analysis

Project ini bertujuan untuk menganalisis sentimen komentar YouTube terhadap isu pelemahan nilai rupiah dan dampaknya terhadap persepsi daya beli masyarakat.

## Topik Project

Analisis Sentimen Komentar YouTube terhadap Isu Pelemahan Nilai Rupiah dan Dampaknya terhadap Persepsi Daya Beli Masyarakat Menggunakan Python, Jupyter Notebook, GitHub, dan Streamlit.

## Latar Belakang Singkat

Pelemahan nilai rupiah terhadap dolar Amerika Serikat sering dikaitkan dengan kenaikan harga barang impor, inflasi, biaya hidup, serta perubahan persepsi masyarakat terhadap daya beli. Komentar YouTube dapat menjadi salah satu sumber data opini publik yang dapat dianalisis menggunakan pendekatan Data Mining dan Natural Language Processing.

## Alur Project

1. Setup Anaconda Environment
2. Setup Jupyter Notebook
3. Setup GitHub Repository
4. Crawling YouTube Comments
5. Data Cleaning
6. Text Preprocessing Bahasa Indonesia
7. Sentiment Labeling
8. Modeling
9. Evaluation
10. Visualization
11. Streamlit Dashboard
12. Deployment to Streamlit Community Cloud
